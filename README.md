# Rocket

Asistente personal de IA con enfoque en gestión financiera.

## Estructura (Monorepo)

- `packages/api` — Servidor backend con Elysia (Bun)
- `packages/web` — Frontend PWA con React + Vite
- `packages/shared` — Tipos y utilidades compartidas

## Desarrollo

1. Instalar dependencias: `bun install`
2. Levantar ambos servicios: `bun dev`
